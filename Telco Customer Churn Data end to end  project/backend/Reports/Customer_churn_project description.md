# 📌 Project Description

The Customer Churn Prediction Project is designed to help businesses identify customers who are likely to stop using their services. By analyzing historical customer data, the project predicts churn based on factors like Age,Gender, contract type, monthly charges, tenure, payment methods, and  Totel charges . Using data preprocessing, outlier detection with SciPy, feature engineering, and machine learning models such as  Random Forest the project provides actionable insights to reduce churn.

The project also includes an interactive Streamlit web app, allowing users to input customer information and receive real-time predictions. This end-to-end solution demonstrates the power of data-driven decision-making to improve customer retention and optimize business strategy.
